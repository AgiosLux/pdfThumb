# pdfThumb

Install the dependncies:

```shell
pip install -r requirements.txt
```

To run the serve, use:

```shell
python app.py
```

## Example of request

```shell
https://pdf-thumb-ijp59ght7-thesilvaemily.vercel.app/get?pdf=https://olacesar.com/e-books/biblia.pdf
```

## Dependencies

* Flask
* PyMuPDF
* request
* Pillow
